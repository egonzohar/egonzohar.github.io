# Ubie Humor — teste de humor

Site estático, responsivo e pronto para GitHub Pages.

## Publicar no GitHub

1. Extraia o arquivo ZIP.
2. Envie **os arquivos internos** (`index.html`, `styles.css`, `app.js`, `favicon.svg`, `.nojekyll`) para a raiz de `egonzohar/egonzohar.github.io`.
3. Confirme o commit na branch `main`.
4. Em **Settings → Pages**, selecione **Deploy from a branch**, branch `main` e pasta `/ (root)`.
5. O endereço será `https://egonzohar.github.io/`.

## Privacidade e funcionamento

- Não há servidor, formulário, analytics ou banco de dados.
- O progresso é salvo apenas no `localStorage` do navegador.
- O resultado é educativo e não oferece diagnóstico.
- Respostas sobre autoagressão acionam orientações de segurança e contatos brasileiros.

## Personalização

- Textos e perguntas: `app.js`
- Cores, tipografia e layout: `styles.css`
- Metadados e estrutura: `index.html`
